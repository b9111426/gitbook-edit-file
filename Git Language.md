# Git語法
## 環境設定
* 輸入作者資訊
```
設定作者E-mail
git config --global user.email "b9111426@gmail.com"
設定作者名稱
git config --global user.name "sausage72"
查詢作者資訊
git config --list  
```
---------------------------------------
* 在本地資料夾新增數據庫
```
git init
```
* 複製遠端數據庫
```
git clone 遠端數據庫網址
```
