# Markdown語法 
###圖片
```
![顯示的敘述][圖片網址]
例如:![Google]: http://google.com/
```

---------------------------------------
###代碼塊
行內`dfasdfds`
```
行內`dfasdfds`
```
---------------------------------------
###連結
```
1.<網址>
範例:<http://example.com/>

2.[描述文字](網址)
範例:[NPM](https://www.npmjs.com/)
```

---------------------------------------
###文字方面
#####<html語法>

1. <font color="red">文字顏色</font>
```
<font color="red">文字顏色</font>
```

2. <table><tr><td bgcolor=85ef47>背景色的字</td></tr></table>
```
<table><tr><td bgcolor=#FF00FF>背景色的字</td></tr><table>
```
3. **粗體字**
```
**bold**
```
4. 標題字
```
# 文子
```
