安裝gitbook

安裝 gitbook

安裝步驟

  1. 安裝Node.js
  2. 通過npm 來安裝gitbook，輸入安裝指令
    npm install gitbook-cli -g

基本使用

  1. gitbook init安裝gitbook

    README.md和SUMMARY.md是兩個必須文件, README.md是對書籍的簡單介紹, SUMMARY.md是書籍的目錄結構

  2. gitbook serve

    gitbook serve命令實際上會首先調用gitbook build編譯書籍，完成以後會打開一個web服務器，監聽在本地的4000端口
